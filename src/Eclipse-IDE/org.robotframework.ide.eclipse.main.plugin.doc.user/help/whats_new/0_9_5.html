RED 0.9.5

Here are some noteworthy features and fixes available in RED 0.9.5

Eclipse platform

Standalone branded RED is using eclipse 2020-06 (4.16) as platform. RED is also dropping support for eclipse Oxygen (4.7) thus making Photon (4.8) the oldest supported platform.

Robot Framework 2.x

Support for Robot Framework 2.9 was dropped. From now only RF 3.x is supported.
